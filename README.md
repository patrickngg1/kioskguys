## The Website
Basically this is the remote version, hosted on Render, here's the link to the [website](https://kioskguys-front.onrender.com/).

## Editing the Website
Any changes made in the repository have to committed and pushed for them to take effect. Since we're currently on the free plan, wait about a minute for changes to take effect. 

## Troubleshooting
If your changes broke something, it won't take effect and will throw an error in the Render console. You are also able to inspect the page and look at the console or network output to see what the issue could be.
